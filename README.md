# student-event-participation-analysis
Power BI project analyzing the impact of campus event participation on student confidence, networking, and soft-skill development using survey data.


# Impact Analysis of Student Participation in Campus Events

## Project Overview
This project analyzes how student participation in campus events influences networking, confidence, and soft-skill development. Survey data was analyzed to uncover trends in participation, motivation, and perceived benefits.

## Tools Used
- Power BI
- Microsoft Excel

## Key Objectives
- Analyze student participation patterns in campus events
- Measure impact on confidence and networking
- Identify motivators and barriers to participation

## Key Features
- Cleaned and structured survey data using Excel
- Interactive Power BI dashboard with slicers and filters
- Visual analysis of confidence levels, social circle growth, and skill development
- Insights on key motivators and challenges faced by students

## Insights
- Higher event participation is positively linked with confidence and networking growth
- Skill-building and networking are major motivators
- Time constraints and stage fear are key participation barriers
